# 머지 충돌 해결하기

_충돌이 발생하는 이유와 해결 방법을 배워보세요._

## 환영합니다

머지 충돌은 두 명이 GitHub에서 같은 파일의 같은 부분을 수정할 때 발생하며, 다른 사람과 협업할 때 흔히 일어나는 일입니다. 차이점을 해결하는 과정에서 논의가 필요할 수 있지만, 머지 충돌은 두려워할 것이 없습니다.

- **대상**: 새로운 개발자, Git 및 GitHub 초보 사용자
- **배울 내용**: 머지 충돌을 확인하고 해결하는 도구를 소개합니다.
- **만들 것**: 웹 편집기를 사용하여 마크다운 파일의 충돌하는 변경 사항을 수정합니다.
- **사전 요구사항**:
  - Skills 실습: [GitHub 소개](https://github.com/skills-kr/introduction-to-github)
- **소요 시간**: 30분 이내

이 실습에서 다음을 수행합니다:

1. 머지 충돌의 원인 배우기
2. 머지 충돌을 해결하는 도구 배우기
3. 머지 충돌의 해결 시점과 방법 배우기
